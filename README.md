Provides some helper classes to handle CSV files easily.

## Usage

### Add the dependency in Gradle

    repositories {
        maven {
            url  "http://dl.bintray.com/joffrey-bion/maven"
        }
    }

    dependencies {
        compile 'org.hildan.utils:csv-utils:0.9'
    }
